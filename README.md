## Descripcion del repositorio 
Sitio web de **turismo** hecho para practicar **HTML** e **Imagenes** con control de versiones en **Git/Github**. Incluye dos paginas: `index.html` y `destinos.html`, una hoja de estilos `style.css` y una carpeta `images/`.
